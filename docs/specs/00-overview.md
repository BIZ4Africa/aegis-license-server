## Project Overview & Objectives

AEGIS is a proprietary license control framework developed by Business Solutions For Africa (BIZ4A).
Its purpose is to protect high-value Odoo modules requiring significant investigation and intellectual property protection.

AEGIS is designed as a dual-component system:
- License Server
- Odoo Client

This repository hosts the server-side component and official specifications.
